    Branching Processes¶

    1. What this topic is about¶

    A branching process models a population where each individual independently produces a random number of offspring. The key question is: does the population survive forever or die out? The answer depends on the mean offspring \(c = E[X]\): if \(c \leq 1\), the process dies out almost surely; if \(c > 1\), there's a positive probability of survival. Branching processes appear everywhere in this course as approximations to early-stage epidemic spread, BFS exploration of random graphs, and component growth.

    2. Core objects / definitions¶

    Branching process \(T_X\): Start with one individual. Each individual independently has \(X\) offspring (drawn from distribution \(D_X\)).

    Generation sizes: \(Z_0 = 1\), \(Z_{n+1} = \sum_{i=1}^{Z_n} X_i\) where \(X_i\) are i.i.d. copies of \(X\).

    Mean offspring: \(c = E[X]\). We have \(E[Z_n] = c^n\).

    Probability generating function (PGF): \(G(\lambda) = E[\lambda^X] = \sum_k p_k \lambda^k\).

    Extinction probability: \(\eta = \Pr(|T_X| < \infty)\) is the smallest non-negative solution of \(G(\eta) = \eta\).

    Survival probability: \(\theta = 1 - \eta = \Pr(|T_X| = \infty)\).

    3. Standard proof moves¶

    1. Compute mean \(c = E[X]\) — this tells you if process is sub/supercritical
    2. Write PGF \(G(\lambda) = E[\lambda^X]\)
    3. Solve \(G(\eta) = \eta\) for extinction probability (take smallest solution in \([0,1]\))
    4. For growth rate: \(E[Z_n] = c^n\), and conditioned on survival, \(\frac{1}{n}\log Z_n \to \log c\)

    4. Important lemmas / facts¶

    Lemma: Extinction criterion¶

    Statement: \(\theta > 0\) iff \(c > 1\).

    Intuition: If each individual produces on average more than one offspring, the population can sustain itself.

    Lemma: Exponential growth¶

    Statement: Conditioned on survival, \(Z_n\) grows exponentially: $\(\Pr(c^{(1-\delta)n} \leq Z_n \leq c^{(1+\delta)n} \mid |T_X| = \infty) \to 1\)$

    Proof outline: - Upper bound: Markov: \(\Pr(Z_n > c^{(1+\delta)n}) \leq c^{-\delta n}\) - Lower bound (the hard part): 1. Apply concentration bound: \(\Pr(Z_{n+1} \leq c' t \mid Z_n = t) \leq e^{-Dt}\) for \(D = I(c')\) 2. Define increasing targets \(k_t = \lceil c^{(1-\delta/2)t} k_0 \rceil\) 3. If \(Z\) ever drops below target, there's a first "bad step" \(s\) where \(Z_{n_0+s} \geq k_s\) but \(Z_{n_0+s+1} < k_{s+1}\) 4. Union bound: \(\sum_{s} e^{-Dk_s} \leq \frac{e^{-Dk_0}}{1-e^{-D}}\) (geometric series since \(k_s \geq k_0 + s\)) 5. Use the bonus result to handle \(Z_{n_0} < k_0\)

    Lemma: Linear growth conditioned on survival¶

    Statement: \(\Pr(Z_n \leq \frac{c-1}{2}n \mid |T_X| = \infty) \leq \frac{e^{-D'n}}{1-e^{-D'}}\) where \(D' = I(\frac{c+1}{2})\).

    Proof idea: If \(Z_n = Y_{t_n}\) (BFS queue at generation boundary), then \(Z_n\) small means \(Y_t\) is small for some \(t \geq n\). Apply concentration + union bound over \(t \geq n\).

    PGF for Poisson offspring¶

    Statement: If \(X \sim \text{Poi}(c)\), then \(G(\lambda) = e^{-c(1-\lambda)}\).

    Extinction equation: \(\eta = e^{-c(1-\eta)}\), equivalently \(\theta + e^{-c\theta} = 1\) (same as giant component equation!).

    PGF for SIR offspring¶

    Statement: \(G_c(\lambda) = \frac{1}{1+c(1-\lambda)}\) (limiting form).

    Extinction equation: \(\eta = 1/c\), so \(P = 1 - 1/c\).

    5. Common mistakes¶

    • Confusing \(\eta\) (extinction) with \(\theta\) (survival): \(\theta = 1 - \eta\)
    • Solving \(G(\eta) = \eta\) but taking the wrong root — always take the smallest in \([0,1]\)
    • Forgetting that exponential growth is conditional on survival — unconditionally, \(Z_n\) could be 0
    • In the lower bound proof: trying to apply concentration directly to \(Z_n\) instead of going through the step-by-step argument with targets \(k_t\)
    • Mixing up \((1-\delta)\) vs \((1-\delta/2)\) in the growth rate — the \(\delta/2\) slack is needed for the inductive argument

    6. Quick memory hooks¶

    • "\(c > 1\) = supercritical = survives; \(c \leq 1\) = dies"
    • "\(E[Z_n] = c^n\) is easy. Showing \(Z_n \approx c^n\) w.h.p. is the real work"
    • "Upper bound is Markov (one line). Lower bound is the union-bound-over-targets argument"
    • "PGF fixed point = extinction probability"

    SIR and Epidemics¶

    1. What this topic is about¶

    The stochastic SIR model describes how an infection spreads through a population. Each infected individual infects susceptible ones at rate \(\beta = c/n\) and recovers at rate \(\gamma = 1\). The key insight is that the early stage of an epidemic looks like a branching process, and the survival probability of that branching process determines whether a large outbreak occurs.

    The main result: for \(c > 1\), a large outbreak happens with probability \(P = 1 - 1/c\). For \(c \leq 1\), the epidemic always dies out.

    2. Core objects / definitions¶

    SIR dynamics: Infected individual \(i\) stays infected for \(T_i \sim \text{exp}(1)\). During that time, infects susceptible \(j\) if infection clock \(T_{ij} \sim \text{exp}(\beta)\) fires before recovery, i.e., \(T_{ij} < T_i\).

    Infection probability given recovery time: $\(p_T = \Pr(T_{ij} < T) = 1 - e^{-\beta T}\)$

    Offspring distribution: Conditioned on \(T\), number infected is \(X \mid T \sim \text{Bin}(n, p_T)\).

    Basic reproduction number: \(R_0 = E[X] = \frac{n\beta}{1+\beta} = \frac{c}{1+c/n} \to c\) as \(n \to \infty\).

    Generating function (limiting): $\(G_c(\lambda) = \frac{1}{1 + c(1-\lambda)}\)$

    Extinction probability: Smallest solution of \(\eta = G_c(\eta)\), which gives \(\eta = 1/c\) for \(c > 1\).

    3. Standard proof moves¶

    1. Condition on recovery time \(T\): Write offspring as \(\text{Bin}(n, p_T)\) conditional on \(T \sim \text{exp}(1)\)
    2. Compute generating function: Integrate over \(T\) to get \(G_n(\lambda)\)
    3. Take \(n \to \infty\) limit: Use dominated convergence to get \(G_c(\lambda)\)
    4. Solve fixed-point equation: \(G_c(\eta) = \eta\) gives extinction probability
    5. For tail bounds below criticality: Use moment generating function + Chernoff-style bounds

    4. Important lemmas / facts¶

    Lemma: Infection probability¶

    Statement: \(p_T = 1 - e^{-\beta T}\) where \(T' \sim \text{exp}(\beta)\).

    Why it matters: This converts the continuous-time infection process into a probability you can plug into a binomial.

    Proof: \(p_T = \Pr(T' < T) = \int_0^T \beta e^{-\beta t'} dt' = 1 - e^{-\beta T}\).

    Lemma: Limiting generating function¶

    Statement: \(G_n(\lambda) \to G_c(\lambda) = \frac{1}{1+c(1-\lambda)}\) for \(\lambda \in [0,1]\).

    Why it matters: Links the finite-population SIR model to a simple branching process.

    Proof idea: 1. Write \(G_n(\lambda) = \int_0^\infty (1-(1-\lambda)p_T)^n e^{-T} dT\) 2. Show \((1-(1-\lambda)p_T)^n \to e^{-cT(1-\lambda)}\) pointwise (since \(np_T \to cT\)) 3. Apply dominated convergence (integrand \(\leq 1\), \(e^{-T}\) integrable) 4. Evaluate: \(\int_0^\infty e^{-T(1+c(1-\lambda))} dT = \frac{1}{1+c(1-\lambda)}\)

    Lemma: Survival probability¶

    Statement: For \(\beta = c/n\): survival probability \(\to 0\) if \(c \leq 1\), and \(\to 1 - 1/c\) if \(c > 1\).

    Proof idea: Solve \(G_c(\eta) = \eta\): $\(\eta(1 + c(1-\eta)) = 1 \iff (1-\eta)(1-c\eta) = 0\)$ Solutions: \(\eta = 1\) and \(\eta = 1/c\). Smallest is \(1/c\) (for \(c > 1\)), so \(P = 1 - 1/c\).

    Lemma: Concentration below criticality¶

    Statement: For \(c < 1\): \(\Pr(|T_{\text{SIR}}^{n,\beta}| > k) \leq e^{-\frac{(1-c)^2}{4}k}\).

    Why it matters: Outbreak size has exponential tail when below threshold.

    Proof idea: \(|T| > k\) implies \(X_1 + \cdots + X_k \geq k\). Apply Chernoff bound with rate function \(I(x) \geq \frac{(x-c)^2}{2x(x+1)}\).

    Lemma: Rate function bound¶

    Statement: For \(c \leq x \leq 1\): $\(I(x) \geq x\log\frac{x}{c} + (1+x)\log\frac{1+c}{1+x} \geq \frac{(x-c)^2}{2x(1+x)}\)$

    Proof idea: 1. Bound MGF: \(E[e^{tX}] \leq \frac{1}{1-c(e^t-1)}\) (using \(p_T \leq \beta T\) and \(1+x \leq e^x\)) 2. Optimize Chernoff bound: \(I(x) \geq \sup_t \{tx + \log(1-c(e^t-1))\}\) 3. Optimal \(t_0 = \log\frac{(1+c)x}{c(1+x)}\) 4. Taylor expand around \(x=c\) to get quadratic lower bound

    5. Common mistakes¶

    • Forgetting to condition on recovery time \(T\) before computing offspring distribution
    • Using \(np_T = c\) instead of \(np_T \to cT\) (the \(T\) matters!)
    • Not checking dominated convergence conditions (need integrand bounded by integrable function)
    • Confusing extinction probability \(\eta\) with survival probability \(P = 1 - \eta\)
    • In the fixed-point equation: \(\eta = 1/c\) is the extinction prob, not \(c\) itself
    • In concentration proofs: the rate function bound requires \(c \leq x \leq 1\), not just \(x \geq c\)

    6. Quick memory hooks¶

    • "Condition on \(T\), then it's just a binomial" — this unlocks everything
    • "\(R_0 = c\) is the threshold, survival prob = \(1 - 1/c\)"
    • "Generating function \(G_c(\lambda) = 1/(1+c(1-\lambda))\) — memorize this"
    • "Below threshold: outbreak dies exponentially fast, rate \((1-c)^2/4\)"
    • "DCT is the magic word for \(G_n \to G_c\)"

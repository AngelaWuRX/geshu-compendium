    Small World Models¶

    1. What this topic is about¶

    Real social networks have two seemingly incompatible features: short typical distances (\(\text{diam} \approx \log n\)) AND high local clustering (\(C_{\text{loc}}\) bounded away from 0). \(G(n,p)\), CM, and PA all satisfy short distances but have \(C_{\text{loc}} \to 0\). Small world models fix this by starting with a network of high local clustering (e.g., a cycle with short-range edges) and adding a few long-range "shortcut" edges that bring the diameter down.

    2. Core objects / definitions¶

    • Local clustering: \(C_{\text{loc}} = \frac1n \sum_i \frac{\#\text{triangles at }i}{\binom{d_i}{2}}\) — fraction of "two friends of \(i\)" who are also friends.
    • Global clustering: \(C_{\text{gl}} = \frac{\sum_i \#\Delta \ni i}{\sum_i \binom{d_i}{2}}\) — different normalization.
    • Diameter (here): typical pairwise distance in \(C_{\max}\) (abuse of notation, but standard in this course).
    • Small world property: \(\text{diam} = O(\log n)\) AND \(C_{\text{loc}} = \Omega(1)\).

    For \(G(n,p), \text{CM}(d), \text{PA}\): \(\text{diam} \asymp \log n\) but \(C_{\text{loc}} \to 0\).

    3. Watts-Strogatz model [1998]¶

    1. Start with cycle \(C_n\). Degrees = 2, \(C_{\text{loc}} = 0\), \(\text{diam} = n/2\).
    2. Connect all vertices at distance \(\le k\) on the cycle. Degrees = \(2k\), \(C_{\text{loc}} > 0\), \(\text{diam} = n/(2k)\).
    3. Rewire each edge with prob \(p\): detach one endpoint, reconnect to a uniform random vertex.

    Numerical observation. As \(p\) increases from 0, diameter drops fast (driven by long-range edges), while \(C_{\text{loc}}\) stays high (most edges stay local).

    Heuristic for diameter (when \(1 \ll kp \ll \text{const}\)): $\(\text{diam} \approx \text{const} \cdot \frac{\log n}{k^2 p}.\)$

    Reasoning: - Long-range rewired edges behave like \(G(n, kp/n)\) → \(\text{diam} \approx \log n / \log(kp)\). - Short-range cycle edges further shorten distances by a factor of \(k\).

    4. Bollobas-Chung [rigorous result]¶

    • Start with \(C_n\).
    • Add one half-edge to each vertex.
    • Match half-edges uniformly at random.

    Theorem. \(\text{diam}(G) / \log_2 n \to_P 1\).

    Why \(\log_2\)? Each vertex has 1 long edge + 2 short edges. Locally exploring follows a 2D random walk on (long, short) types with transition matrix \(\Pi = \begin{pmatrix} 0 & 2 \\ 1 & 1 \end{pmatrix}\), so BFS tree grows at rate \(\nu = \lambda_{\max}(\Pi) = 2\).

    5. Newman-Watts [1999] + Barbour-Reiner [2006]¶

    • Local graph as in WS (cycle + short range).
    • Don't rewire — instead add global edges from \(G(n, c/n)\).

    Barbour-Reiner Theorem. If \(c \gg 1\) and \(kc \le \text{const}\): $\(\frac{\text{diam}(G)}{\log_\nu n} \to_P 1, \quad \nu = \lambda_{\max}\begin{pmatrix} c & c \\ 2k & 1\end{pmatrix} = \tfrac12\big[c+1 + \sqrt{(c+1)^2 + 4c(2k-1)}\big].\)$

    Limit case (\(kc \ll 1\), \(\nu \approx 1 + 2k\lambda\)): $\(\text{diam}(G) \approx \frac{\log n}{2k\lambda}.\)$

    This matches the WS heuristic with \(p \mapsto \lambda\).

    6. Kleinberg navigation [2001-02] — the classic result¶

    Setup (Milgram '67). Can a "myopic agent" find a target \(t\) in \(O(\log n)\) steps?

    Myopic agent rules: - Can follow short or long edges. - Sees one step ahead. - Knows the physical location of target + possible next moves.

    Greedy on cycle with one long random edge per vertex¶

    Theorem 1. Greedy needs \(\Theta(\sqrt n)\) steps. Theorem 2. No myopic algorithm does better.

    Proof idea (Theorem 1). It takes \(\Theta(\min\{r, n/r\})\) steps to reduce distance from \(r/2\) to \(r\). Set \(r_i = n \cdot 2^{-i}\) for \(i = 1, \ldots, T = \log_2 n\). Total: \(\sum_{i=1}^T \min\{n 2^{-i}, 2^i\} = \Theta(\sqrt n)\).

    Modified small world model on \(\{0, \ldots, L-1\}^d\) (\(n = L^d\))¶

    • Add local edges for \(\text{dist} \le p\).
    • Each vertex chooses \(q\) long-range edges to \(u_1, \ldots, u_q\) with prob \(\Pr(v \to u_i) \propto 1/d(v, u_i)^\alpha\).

    Theorem 3 (Kleinberg). - If \(\alpha = d\) (the "magic" exponent): greedy takes \(\Theta((\log n)^2)\) steps. - If \(\alpha \ne d\): any myopic algorithm takes \(n^c\) steps for some \(c > 0\).

    Proof idea. - \(\alpha > d\): too local, \(E[\text{dist}(v,u)] \approx \int r^{d-1} r^{-\alpha} dr\) converges → can't get far enough. - \(\alpha < d\): too random → jumping around blindly, no learning. - \(\alpha = d\): just right — every distance scale is equally likely → can halve distance in \(O(\log n)\) rounds.

    7. Household / motif models¶

    • Household models [Trapman '07, Ball '10]: start with \(G\) with \(\text{diam} \approx \log n\), replace nodes by households (cliques or other small graphs).
    • Motif models [Hofstad '15]: start with motifs \(M = (H, d^{\text{ext}})\) — small graphs with external stubs. Match stubs uniformly.

    These are how you get both \(\text{diam} = O(\log n)\) AND \(C_{\text{loc}} = \Theta(1)\) without giving up rigor.

    8. Common mistakes¶

    • Confusing adding edges (Newman-Watts) with rewiring (Watts-Strogatz). Rigor exists for NW; WS is mostly numerical/heuristic.
    • Assuming the WS diameter formula \(\log n / (k^2 p)\) is rigorous — it's a heuristic.
    • For Kleinberg: \(\alpha = d\) (dimension), not \(\alpha = 1\). In 2D grid, the magic exponent is 2.
    • Greedy in 1D Kleinberg model takes \(\Theta(\sqrt n)\), not \(\Theta(\log n)\) — the magic exponent for 1D requires \(\alpha = 1\).

    9. Quick memory hooks¶

    • "Local clustering + short diameter ⇒ need long-range edges."
    • "Newman-Watts: keep the cycle, sprinkle in \(G(n,c/n)\) on top."
    • "Kleinberg: \(\alpha = d\) is the only exponent that lets greedy navigation be polylog."
    • "Watts-Strogatz heuristic: \(\log n / k^2 p\) — long edges shorten by \(\log n / \log(kp)\), short edges shorten by another factor of \(k\)."

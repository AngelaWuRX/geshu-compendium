    Stochastic Block Model and Multitype Branching Process¶

    SBM definition¶

    \(Q=[q]\) types/colors, type fractions \(\alpha=(\alpha_k)\), symmetric \(B=(B_{k\ell})\in\mathbb R_+^{q\times q}\).

    \(\text{SBM}_n(B,\alpha)\). Each \(i\in[n]\) independently draws color \(k(i)\sim\alpha\). Conditional on colors, edges are independent with \(\Pr(ij\in E) = B_{k(i),k(j)}/n\).

    ⇒ A vertex of color \(k\) has \(\text{Bin}(n_\ell, B_{k\ell}/n) \approx \text{Poi}(B_{k\ell}\alpha_\ell) =: \text{Poi}(C_{k\ell})\) neighbors of color \(\ell\). Mean matrix \(C = (C_{k\ell})\).

    Multitype Branching Process (MTBP)¶

    Type-\(i\) parent has \(\vec X_i = (X_{i1},\dots,X_{iq})\) children, with \(X_{ij}\) = #type-\(j\) children. Joint PGF $\(G_i(\vec\lambda) = \sum_{\vec k}\Pr(\vec X_i=\vec k)\prod_j\lambda_j^{k_j}.\)$ Mean matrix \(C_{ij}=\mathbb E[X_{ij}]\).

    SBM example. \(X_{ij}\sim\text{Poi}(C_{ij})\) independent ⇒ \(G_i(\vec\lambda) = \exp\Big(\sum_j C_{ij}(\lambda_j-1)\Big)\).

    Generation-\(t\) mean¶

    Let \(Z_{ij}(t)\) = #type-\(j\) descendants in generation \(t\) from a type-\(i\) root. $\(\mathbb E[Z_{ij}(t+1)\mid \vec Z_i(t)] = \sum_\ell \mathbb E[X_{\ell j}]Z_{i\ell}(t) \;\Rightarrow\; \mathbb E[Z_{ij}(t)] = (C^t)_{ij}.\)$

    Reducibility / irreducibility¶

    \(B\) is reducible iff \(\exists k\neq\ell\) with \((B^t)_{k\ell}=0\) \(\forall t\) (no path between types). Equivalent: types cluster into groups with no inter-group paths. Irreducible = otherwise.

    Lemma (Subcritical: extinction)¶

    If \(\lambda_{\max}(C) < 1\), then \(\eta_i = \Pr(\text{extinction}\mid\text{root type }i) = 1\) (\(\theta_i = 0\)).

    Proof. \(\Pr(\vec Z_i(t)\neq 0) \leq \mathbb E[\sum_j Z_{ij}(t)] = \sum_j(C^t)_{ij} \leq O(t^{q-1}\lambda_{\max}^t) \to 0\). (For symmetric \(C\), drop the \(t^{q-1}\) factor.) So \(\theta_i = \lim_{t\to\infty}\Pr(\vec Z_i(t)\neq 0) = 0\).

    Lemma 1 (Iterated PGF gives extinction-by-\(t\))¶

    \(\eta_i(t) = \Pr(\vec Z_i(t)=0)\) satisfies \(\eta_i(t+1) = G_i(\vec\eta(t))\), \(\vec\eta(0)=\vec 0\).

    Proof (induction). \(t=0\): \(\eta_i(1) = \Pr(\vec X_i=\vec 0) = G_i(\vec 0)\). Step: condition on first generation \(\vec X_i = \vec k\). Each of the \(\sum k_j\) children initiates an indep copy of the BP; all must die in \(t\) generations: $\(\Pr(\vec Z_i(t+1)=0\mid \vec X_i=\vec k) = \prod_j \eta_j(t)^{k_j}.\)$ Sum over \(\vec k\) weighted by \(P_i(\vec k)\) gives \(G_i(\vec\eta(t))\).

    Lemma 2 (Monotonicity)¶

    \(\vec\eta(t)\uparrow\vec\eta\) where \(\eta_i = \Pr(\text{extinction})\).

    (Events \(\{Z_i(t)=0\}\) nested ⇒ probabilities monotone; bounded by 1.)

    Theorem (Fixed-point characterization)¶

    (a) \(\eta_i(t)\uparrow\eta_i\). (b) \(\vec\eta = \vec G(\vec\eta)\). (c) \(\vec\eta\) is the smallest fixed point in \([0,1]^q\): if \(\vec G(\vec\lambda)=\vec\lambda\) then \(\vec\eta\leq\vec\lambda\).

    Proof. (a) Lemmas 1+2. (b) \(G_i\) monotone+continuous in coordinates ⇒ \(\eta_i(t+1)=G_i(\vec\eta(t))\uparrow G_i(\vec\eta)\), but also \(\eta_i(t+1)\uparrow\eta_i\) ⇒ \(\eta_i = G_i(\vec\eta)\). (c) \(\vec G(\vec\lambda)=\vec\lambda\), \(\vec 0\leq\vec\lambda\): \(\eta_i(1)=G_i(\vec 0)\leq G_i(\vec\lambda)=\lambda_i\). Iterate: \(\eta_i(t)\leq\lambda_i\) ⇒ \(\eta_i\leq\lambda_i\).

    Theorem (Survival ⇔ supercritical)¶

    For irreducible, non-singular MTBP: \(\theta_i = 1-\eta_i > 0\) for all \(i\) ⇔ \(\lambda_{\max}(C) > 1\).

    (Singular = each parent has exactly one child a.s., trivially never extinct.)

    Application to SBM (giant component)¶

    Theorem (Giant component on SBM)¶

    Assume \(C\) irreducible. (a) If \(\lambda_{\max}(C) < 1\): w.h.p. \(|C_{\max}| = O(\log n)\). (b) If \(\lambda_{\max}(C) > 1\): \(\exists\theta>0\) with \(|C_{\max}|/n \to \theta\) in probability, where $\(\theta = \sum_i \alpha_i\theta_i,\qquad \theta_i = 1 - G_i(\vec\eta) = 1 - e^{-\sum_j C_{ij}\theta_j}\;\text{(SBM)}.\)$ Off-giant components: \(\max_{x\notin C_{\max}}|C(x)| = O(\log n)\).

    Proof sketch. 1. BFS coupling. Local exploration of SBM at a type-\(i\) vertex couples to MTBP\((C)\) for \(O(\log n)\) generations. 2. Subcritical. All BFS trees stay tiny ⇒ tail bound on \(|C(x)|\) via mean of \(\sum_j(C^t)_{ij} = O(t^{q-1}\lambda_{\max}^t)\). 3. Supercritical. Define \(V_i^+ = \{x: k(x)=i, |C(x)|>K\log n\}\). BFS gives \(|V_i^+|/n \to \theta_i\alpha_i\) in probability. A sprinkling argument (small-tree lemma + connectivity of "big" pieces above a threshold) shows all of \(\bigcup_i V_i^+\) merges into a single giant: \(||C_{\max}|/n - \sum_i\alpha_i\theta_i| \to 0\).

    Pitfalls¶

    • Use \(\lambda_{\max}(C)\), not the entries of \(C\), as the criticality test.
    • Smallest fixed point of \(\vec G(\vec\eta)=\vec\eta\) is the extinction prob; survival is \(1\) minus that.
    • For SBM with type fractions: \(\theta = \sum_i \alpha_i\theta_i\) (weight by type fraction).

    Information Cascades and Influence Maximization¶

    Models¶

    Linear Threshold (LT). Weights \(b_{uv}\geq 0\), thresholds \(\theta_v\in[0,1]\) uniform i.i.d. at \(t=0\). Active set evolves $\(A_t = A_{t-1}\cup\Big\{v\notin A_{t-1}:\sum_{w\in A_{t-1}: wv\in E} b_{wv}\geq\theta_v\Big\}.\)$

    Independent Cascade (IC). Probabilities \(p_{uv}\). When \(u\) first becomes active, it gets one chance to activate each inactive out-neighbor \(v\) with probability \(p_{uv}\) (independent).

    Influence: \(I(A) = \mathbb E[\lim_t |A_t| \mid A_0=A]\).

    Influence-Max problem: Find \(A^*\) with \(|A^*|=k\) maximizing \(I(A)\). NP-hard (reduction from Vertex Cover, KKT'03).

    Submodularity¶

    Def. \(f:2^V\to\mathbb R^+\) is submodular if for all \(v\in V\), \(S\subseteq T\subseteq V\): $\(f(S\cup\{v\})-f(S) \;\geq\; f(T\cup\{v\})-f(T).\)$ ("Diminishing returns".) Monotone: \(f(S\cup\{v\})\geq f(S)\).

    Lemma 1 (Convex combinations preserve submodularity)¶

    If each \(f_i\) is monotone+submodular and \(p_i\geq 0\), then \(f=\sum_i p_i f_i\) is too.

    Proof. \(f(S\cup v)-f(S)=\sum_i p_i[f_i(S\cup v)-f_i(S)] \geq \sum_i p_i[f_i(T\cup v)-f_i(T)] = f(T\cup v)-f(T)\).

    Lemma 2 (IC influence is submodular)¶

    Random digraph reduction. Pre-flip all coins \(X_{uv}\sim\text{Be}(p_{uv})\); let \(E(X)\) = open edges. The cascade from seeds \(A\) activates exactly \(C(A) = \bigcup_{v\in A}C(v)\) where \(C(v)\) = vertices reachable from \(v\) in \(D(X)=(V,E(X))\). Then \(I(A) = \mathbb E_X[|C_X(A)|]\).

    Submodularity of \(|C_X(\cdot)|\) (each fixed \(X\)). Fix \(S\subseteq T\). The "new" reachable set when adding \(v\) is \(C_X(v)\setminus C_X(S)\). Since \(C_X(S)\subseteq C_X(T)\): $\(|C_X(S\cup v)| - |C_X(S)| = |C_X(v)\setminus C_X(S)| \geq |C_X(v)\setminus C_X(T)| = |C_X(T\cup v)|-|C_X(T)|.\)$

    Take expectation over \(X\) — Lemma 1 finishes.

    Theorem (Greedy is \((1-1/e)\)-approximation)¶

    For monotone submodular \(f\) with \(f(\emptyset)=0\), the greedy algorithm - Start \(A_0=\emptyset\). - \(A_t = A_{t-1}\cup\{v^*\}\) where \(v^*=\arg\max_{v\notin A_{t-1}} f(A_{t-1}\cup\{v\})\).

    produces \(A_k\) with $\(f(A_k) \geq \big(1-(1-1/k)^k\big)\max_{|A|=k}f(A) \geq (1-1/e)\,\text{OPT}.\)$

    Step 1 (Key claim).¶

    For \(|S|\leq k\) and \(S^*\) optimal of size \(k\): $\(\max_{x\notin S}\big[f(S\cup\{x\})-f(S)\big] \geq \tfrac{1}{k}\big[f(S^*)-f(S)\big].\)$

    Proof. Let \(S^*\setminus S = \{v_1,\dots,v_p\}\), \(p\leq k\). Telescoping: $\(f(S^*) \leq f(S\cup S^*) = f(S) + \sum_{i=1}^p\big[f(S\cup\{v_1,\dots,v_i\}) - f(S\cup\{v_1,\dots,v_{i-1}\})\big].\)$ Submodularity bounds each term by \(f(S\cup\{v_i\})-f(S)\): $\(f(S^*) - f(S) \leq \sum_i [f(S\cup v_i)-f(S)] \leq k\max_{x\notin S}[f(S\cup x)-f(S)].\)$

    Step 2 (Greedy unrolling).¶

    With \(S=A_{t-1}\), greedy picks the maximizer: $\(f(A_t) \geq f(A_{t-1}) + \tfrac{1}{k}[f(A^*)-f(A_{t-1})] = \tfrac{1}{k}f(A^*) + (1-\tfrac{1}{k})f(A_{t-1}).\)$ Iterate from \(A_0=\emptyset\): $\(f(A_k) \geq \tfrac{1}{k}f(A^*)\sum_{i=0}^{k-1}(1-\tfrac{1}{k})^i = f(A^*)\big(1-(1-\tfrac{1}{k})^k\big) \geq (1-1/e)f(A^*).\)$ (Used \(\sum_{i=0}^{k-1}(1-1/k)^i = k(1-(1-1/k)^k)\) and \((1-1/k)^k\leq e^{-1}\).)

    Pitfalls¶

    • Submodularity proof: must condition on \(X\) (the random digraph) first; otherwise \(C(\cdot)\) is random.
    • Greedy gives \(1-1/e\approx 0.632\), not exact optimum (NP-hard).
    • LT is also submodular (KKT'03), but proof is harder (uses a different randomization — not covered).

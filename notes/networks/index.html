    Networks¶

    Eighteen notes on random graphs, and about thirty methods holding them together. The notes are the details; this page is the shape.

    Most of these topics are the same few ideas wearing different clothes. A branching process explains the giant component, then explains an epidemic, then explains the bow-tie in a directed graph. Concentration is how you turn "on average" into "almost surely" everywhere it appears. The tables below are an attempt to make that reuse visible: which method powers which topic, and which question shape reaches for what.

    Start with the decision tree if you have a problem in front of you, or the topic index if you want the map.

    Topics in scope (18 notes): Thresholds, Concentration, Branching Processes, Expected Counts & Union Bounds, Graph Counting, Configuration Model, Preferential Attachment, SIR & Epidemics, Giant Component, Diameter of Giant, SIR on Networks, Small World, Directed Networks, SBM & Multitype BP, Markov Chains & PageRank, Mixing Time & Spectral Gap, Spectral Clustering, Information Cascades.

    1. Master Method Table¶

    # Method Recognition signal First move Topic notes Common trap
    1 First moment / Expected count Show structure rare/absent \(X=\sum I_i\), compute \(E[X]\), show \(\to 0\) Thresholds, Expected Counts Overcounting candidates (cycle orientations, starting points)
    2 Second moment Show structure exists w.h.p. \(E[X^2]\), split \(i=j\) vs \(i\ne j\), ratio \(\to 1\) Thresholds, Expected Counts Missing correlation from shared edges (exponent \(2n-3\))
    3 Markov's inequality Quick upper bound on \(\Pr(X\ge a)\) \(\Pr(X\ge a)\le E[X]/a\) Concentration Using when Chebyshev/Chernov needed
    4 Chebyshev "Close to mean" + variance available \(\text{Var}/t^2\), often via \(\text{Var}\ll E[X]^2\) Concentration Wrong variance from missing overlap cases
    5 Chernov / rate function Sum of i.i.d., need exponential tail MGF + Markov on \(e^{tS}\), optimize \(t\) Concentration, BP Rate function only positive when \(\lambda\ne c\)
    6 Multiplicative Chernov Tail bound relative to \(E[S_k]\) Same recipe; for \([0,1]\) use convexity bound on MGF Concentration, Giant Component, SIR Networks Skipping the \(t\)-optimization step
    7 Union bound "For all vertices/pairs/cycles, bad event doesn't happen" List bad events, sum probabilities Expected Counts Forgetting to sum over all bad events
    8 Threshold reasoning "When does property appear/disappear?" Find critical scaling, prove both directions Thresholds Only proving one direction
    9 Branching-process approximation Early-stage tree-like exploration, epidemic start Identify offspring distribution + mean \(c\) Branching Processes, SIR Using BP beyond its valid regime (\(t \ll \log n\))
    10 Generating function / fixed point Extinction/survival of BP \(G(\lambda)=E[\lambda^X]\), solve \(G(\eta)=\eta\), smallest root Branching Processes, SIR Taking wrong root (need smallest in \([0,1]\))
    11 Multitype PGF / fixed point Multi-type / SBM BP \(\vec\eta = \vec G(\vec\eta)\), smallest fixed point in \([0,1]^q\), criticality \(\lambda_{\max}(C)>1\) SBM & Multitype BP Using entries of \(C\) instead of \(\lambda_{\max}(C)\)
    12 Conditional expectation + recurrence Quantity evolves one step at a time \(E[D(t+1)\mid \mathcal F_t]\) in terms of \(D(t)\) Preferential Attachment Forgetting denominator changes with \(t\)
    13 ODE / fluid-limit method Discrete process on \(n\) objects, want continuous approx Rescale by \(n\), replace differences by derivatives Configuration Model, Giant Component Confusing \(\tilde Y_t\) (can go neg) vs \(Y_t\) (stops at 0)
    14 BFS exploration → fluid sandwich Giant component / outbreak size proof \(\tilde Y_{t+1}=\tilde Y_t+X_{t+1}-1\), get binomial \(\Delta_t\), mult Chernov, sandwich \(f_{\pm\epsilon}\) Giant Component, SIR on Networks Using additive Chernov when multiplicative is needed; forgetting \(\ge K\log n\) conditioning
    15 Two-layer Chernov Process step has both random "type" and inner random outcome Condition on types, get inner binomial; concentrate sum-of-types separately SIR on Networks Trying to do everything in one layer (loses Chernov hook)
    16 Matching probability (config model) Half-edge pairings \(E[m_{\alpha,\beta}] = 1/(\ell_n-1)\); disjoint pairs \(1/[(\ell_n-1)(\ell_n-3)]\) Configuration Model Using \((\ell_n-1)^2\) for two disjoint pairs
    17 Indicator decomposition Expand \(A_{ij}(A_{ij}-1)\) in config model Write in matching indicators \(m_{\alpha,\beta}\) Configuration Model Not exploiting perfect-matching constraint
    18 Graph counting (cycles/trees/wedges) Need # of substructures Overcounted version ÷ symmetries; \((k-1)!/2\) cycles, \(k^{k-2}\) trees, \((M-1)!!\) matchings Graph Counting Tricks Cycles: divide by \(2k\)
    19 Overlap / variance decomposition \(E[X^2]\) for count of substructures Classify pairs by # shared edges/vertices Expected Counts, Graph Counting Missing an overlap case
    20 Dominated convergence (DCT) Integrand converges pointwise, need integral limit Find dominating integrable function SIR & Epidemics Forgetting to verify domination
    21 Coupling Bound TV distance between two chains Construct shared randomness; \(\Pr(X_t\ne \tilde X_t)\) bounds TV Markov Chains & PageRank Coupling not "faithful" — must respect both marginals
    21b Overlap-method MC contraction Direct convergence of \(\mu_t \to \pi\) when \(\min M_{ij} \ge \epsilon\) \(\alpha(u,v) = \sum\min\{u_i,v_i\}\), decompose \(u = \alpha w + (1-\alpha)u'\) Markov Chains & PageRank Just bounding \(\alpha(uM, vM) \ge \alpha\) — need strict \(+(1-\alpha)\epsilon n\)
    21c TV dual / operator-norm form Compose mixing rates across multiple steps \(\tilde d(t) = \sup_f \delta(M^t f)/\delta(f)\), then \(\tilde d(s+t) \le \tilde d(s)\tilde d(t)\) Mixing Time Forgetting to center \(f\) before bounding by $\
    21d Forward/backward SIR BP on CM Outbreak probability vs susceptibility on configuration model Condition on \(T\), write Bin offspring; root uses \(G\), descendants use \(G^*\); back-edges absorb \(T\) into \(\bar p\) SIR on Networks Using \(G\) instead of \(G^*\) for descendants ⇒ wrong \(R_0\)
    22 Spectral / Rayleigh quotient Eigenvalue characterization of \(\lambda_2\) or \(\Lambda_2\) Variational form: \(\min_{\varphi\perp\varphi_1}\frac{\varphi^T\tilde L\varphi}{\varphi^T\varphi}\) Mixing Time, Spectral Clustering Using \(P\) instead of symmetrized \(\tilde P\)
    23 Test-vector technique Bound \(\lambda_2\) from above by exhibiting a test vector Define \(f_S = D^{1/2}\psi_S\) via 2-level \(\psi_S = \pi(S^c)\mathbf 1_S - \pi(S)\mathbf 1_{S^c}\) Mixing Time, Spectral Clustering Forgetting orthogonality \(\psi^T D\mathbf 1 = 0\)
    24 Cheeger inequality Mixing time / conductance link \(\phi^2/8 \le 1-\lambda_2 \le \phi\) Mixing Time Wrong constant; mixing \(\asymp \frac{1}{1-\lambda_2}\log\frac{1}{\pi_{\min}}\)
    25 Spectral decomposition for mixing Convergence rate of \(\mu_t \to \pi\) \(\mu_{t,x}(v) - \pi(v) = \sum_{i\ge 2}\tilde\lambda_i^t \varphi_i(x)\varphi_i(v)\sqrt{d_v/d_x}\) Mixing Time Use lazy chain or only \(\lambda_2\) matters
    26 Sweep-cut algorithm Find a low-conductance cut from \(\psi_2\) Sort by \(\psi_2\), take prefix that minimizes \(\phi(S_k)\) Spectral Clustering Sign-cut isn't always optimal — sweep is
    27 Submodularity + greedy "Choose \(k\) seeds maximizing spread" Show \(f\) submodular monotone; greedy gives \((1-1/e)\) approx Information Cascades Submodularity proof must condition on the random digraph \(X\) first
    28 Random-digraph reduction (IC) Independent Cascade influence Pre-flip \(X_{uv}\sim\text{Be}(p_{uv})\); $I(A) = E_X $ (lecture L27)

    3. Decision tree by question type¶

    What kind of question is this?
│
├── "Show structure doesn't exist w.h.p."         → 1st moment   → Thresholds / Expected Counts
├── "Show structure exists w.h.p."                → 2nd moment   → Thresholds / Expected Counts
├── "X is close to its mean w.h.p."
│   ├── Variance available?                       → Chebyshev    → Concentration
│   ├── i.i.d. sum?                               → Chernov      → Concentration
│   └── Tail bound *relative to* $E[X]$?          → Mult Chernov → Concentration
│
├── "Survival/extinction probability"
│   ├── Single type?                              → $G(\eta)=\eta$, smallest root → BP / SIR
│   └── Multiple types (SBM)?                     → $\vec\eta=\vec G(\vec\eta)$,
│                                                   $\lambda_{\max}(C)>1$ → SBM
│
├── "Quantity evolves over time"
│   ├── Discrete, one step at a time              → Conditional E + recurrence → PA
│   └── Large scale, continuous limit             → ODE / fluid limit → CM, Giant
│
├── "Size of giant / outbreak"
│   ├── ER $G(n,c/n)$                            → BFS sandwich → Giant Component
│   ├── ER + SIR ($R_0=c\bar p$)                 → Two-layer Chernov → SIR on Networks
│   └── CM + SIR                                 → $\theta$-reduction ODE → SIR on Networks
│
├── "How fast does the chain mix?"
│   ├── Have $\lambda_2$ / $\Lambda_2$?          → spectral bound → Mixing Time
│   ├── Have a bottleneck cut?                   → Cheeger ($\phi$) → Mixing Time
│   └── PageRank with restart?                   → coupling, $(1-\alpha)^t$ → MC & PageRank
│
├── "Find communities / partition"
│   ├── Two blocks, planted                      → check $ab>\varepsilon^2$ → Spectral Clustering
│   └── $k$-way clustering                       → top-$d$ eigvecs + $k$-means → Spectral Clustering
│
├── "Choose $k$ seeds, max spread"               → submodular + greedy → Info Cascades
│
└── "Probability of specific multigraph in CM"   → matching indicators + counting → Config Model

    5. Quick formula sheet¶

    Random graph thresholds¶

    • Isolated vertices: \(p^* = \log n / n\)
    • Connectivity: same threshold, \(p^* = \log n / n\)
    • Cycles appear: \(p^* = 1/n\)
    • Giant component: \(c = 1\), i.e. \(p^* = 1/n\)
    • Diameter: \(\Theta(\log n / \log(np))\)

    Counting¶

    • Unoriented \(k\)-cycles on \(k\) labeled vertices: \(\frac{(k-1)!}{2}\)
    • Labeled trees on \(k\) vertices (Cayley): \(k^{k-2}\)
    • Perfect matchings of \(M\) items: \((M-1)!!\)
    • \(\binom{n}{k} \le n^k/k!\), \(k^k \le k! e^{k-1}\)

    Branching / SIR¶

    • Poisson BP PGF: \(G(\lambda) = e^{-c(1-\lambda)}\), extinction \(\eta + e^{-c(1-\eta)}=\eta\) ⇒ \(\theta + e^{-c\theta}=1\)
    • SIR limiting PGF: \(G_c(\lambda) = \frac{1}{1+c(1-\lambda)}\), extinction \(\eta = 1/c\), survival \(P = 1 - 1/c\)
    • Subcritical SIR tail: \(\Pr(|T_\text{SIR}|>k) \le e^{-(1-c)^2 k/4}\)
    • Rate function: \(I(x) \ge \frac{(x-c)^2}{2x(1+x)}\) for \(c\le x\le 1\)

    Multiplicative Chernov¶

    • \(\Pr(S_k \ge (1+\epsilon) E S_k) \le e^{-\epsilon^2 E S_k / (2(1+\epsilon))}\)
    • \(\Pr(S_k \le (1-\epsilon) E S_k) \le e^{-\epsilon^2 E S_k / 2}\)

    Giant component / SIR on \(G(n,p)\)¶

    • \(G(n,c/n)\): \(|C(x)|/n \to \theta\), \(\theta+e^{-c\theta}=1\)
    • SIR on \(G(n,p)\) with recovery \(T\): \(R_0 = c\bar p\), \(\bar p = E[1-e^{-\beta T}]\), \(\theta+e^{-R_0\theta}=1\)
    • Fluid limit: \(f(x) = 1 - e^{-cx} - x\), \(f_{\pm\epsilon}\) sandwich

    Configuration model¶

    • \(E[m_{\alpha,\beta}] = 1/(\ell_n-1)\), disjoint pairs: \(1/[(\ell_n-1)(\ell_n-3)]\)
    • \(E[\text{loops}] = \frac{1}{\ell_n-1}\sum_i\binom{d_i}{2}\)
    • Graph prob: \(P(G) = \frac{1}{(\ell_n-1)!!}\prod_i\frac{d_i!}{2^{A_{ii}}A_{ii}!}\prod_{i<j}\frac{1}{A_{ij}!}\)

    Preferential attachment¶

    • \(E[D_i(t)] \approx D_i(i)\sqrt{t/i}\) (after solving \(\prod(1+1/(2s))\))

    Random walks / mixing¶

    • Stationary on \(G\): \(\pi_v = d_v/(2|E|)\)
    • Symmetrized: \(\tilde P = D^{-1/2}AD^{-1/2}\)
    • PageRank coupling: \(\|\mu_{t,x} - \pi\|_{TV} \le (1-\alpha)^t \le e^{-\alpha t}\)
    • Overlap-method: if \(\min M_{ij} \ge \epsilon\), \(d_{TV}(\mu_t, \pi) \le (1-\epsilon n)^t\)
    • Cheeger: \(\phi^2/8 \le 1-\lambda_2 \le \phi\)
    • Lazy mixing: \(\tilde\lambda_2^t \le \bar d(t) \le \tilde\lambda_2^t / \sqrt{\pi_{\min}}\)
    • Submultiplicativity: \(\tilde d(t+s) \le \tilde d(t) \tilde d(s)\), \(d(k\tau) \le e^{-k}\)
    • Mixing time: \(\tau \asymp \frac{1}{1-\lambda_2}\log(1/\pi_{\min})\)
    • TV three forms: \(d_{TV}(\mu, \pi) = \sup_S |\mu(S)-\pi(S)| = \tfrac12 \|\mu-\pi\|_1 = \sup_f |E_\mu f - E_\pi f|/\delta(f)\)

    Diameter (Giant Component)¶

    • \(G(n, c/n)\): \(D / \log_c n \to_P 1\) for \(c > 1\)
    • CM: \(D / \log_\nu n \to_P 1\) where \(\nu = E[D(D-1)]/E[D]\) (excess degree)
    • Newman-Watts (cycle + \(G(n, c/n)\)): \(D / \log_\nu n \to 1\), \(\nu = \tfrac12[c+1+\sqrt{(c+1)^2+4c(2k-1)}]\)

    Small world (Kleinberg)¶

    • 1D ring + random long edges: greedy takes \(\Theta(\sqrt n)\) steps (no algo does better)
    • \(d\)-D grid + long edges \(\propto 1/d^\alpha\): greedy is \(\Theta((\log n)^2)\) iff \(\alpha = d\), else \(n^c\)

    Directed networks (bow-tie)¶

    • \(D(n, c/n)\), \(c > 1\): SCC core size \(\theta^2 n\), in/out fans \(\theta(1-\theta) n\), rest \((1-\theta)^2 n\)
    • Where \(\theta\) = survival prob of \(\text{Poi}(c)\) BP (same as undirected giant)

    Configuration model — simplicity (Lecture 11)¶

    • \(\nu = E[D(D-1)]/E[D]\)
    • \(\#\text{loops} \to \text{Poi}(\nu/2)\), \(\#\text{multi-edges} \to \text{Poi}(\nu^2/4)\)
    • \(\Pr(G \text{ simple}) \to e^{-\nu/2 - \nu^2/4}\)
    • Chung-Lu: \(p_{ij} = w_i w_j / \ell_n\), giant exists iff \(\tilde c = \sum w_i^2 / \sum w_i > 1\)

    Spectral clustering¶

    • Generalized eigenproblem: \((D-A)\psi = \Lambda D\psi\), \(\Lambda = 1-\lambda\)
    • Variational \(\Lambda_2\): \(\min_{\psi^T D\mathbf 1 = 0} \frac{\psi^T(D-A)\psi}{\psi^T D\psi}\)
    • Two-block recovery: \(ab > \varepsilon^2\)

    Influence max¶

    • IC influence: \(I(A) = E_X|C_X(A)|\) (random digraph reduction)
    • Greedy: \(f(A_k) \ge (1-1/e)\,\text{OPT}\)

    Multitype BP / SBM¶

    • Mean matrix \(C_{k\ell} = B_{k\ell}\alpha_\ell\)
    • Criticality: \(\lambda_{\max}(C) > 1\) ⇔ giant exists
    • Survival: \(\theta = \sum_i \alpha_i\theta_i\), \(\theta_i = 1 - e^{-\sum_j C_{ij}\theta_j}\) (SBM Poisson)

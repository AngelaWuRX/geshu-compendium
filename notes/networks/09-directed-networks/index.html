    Directed Networks (Digraphs, SCC, Bow-tie)¶

    1. What this topic is about¶

    Real networks (web links, citations, retweets) are directed. The component structure is more subtle than the undirected case: instead of one giant connected component, you get a bow-tie: a central giant strongly-connected component (SCC), an in-fan that can reach the SCC, and an out-fan that the SCC can reach. The relevant model is \(D(n, p)\) (random directed Erdős–Rényi), and the analysis reduces to two coupled BFS-trees: forward (out-component) and backward (in-component).

    2. Core objects / definitions¶

    • Digraph \(D = (V, E)\): each ordered pair \((i, j)\) may or may not be an edge; cycles allowed.
    • DAG (Directed Acyclic Graph): digraph with no directed cycles.
    • Strongly connected component (SCC): maximal \(C \subseteq V\) such that for all \(x, y \in C\), both \(\exists\) paths \(x \to y\) and \(y \to x\) in \(C\).
    • Out-component \(C_+(x) = \{y : \exists \text{ directed path } x \to y\}\).
    • In-component \(C_-(x) = \{y : \exists \text{ directed path } y \to x\}\).

    Lemma (DAG of SCCs). Contracting each SCC to a single vertex gives a DAG \(D'\). (Any two SCCs with paths in both directions would merge into a single SCC.)

    3. Models¶

    \(D(n, p)\) — random digraph (Karp '90)¶

    Each of the \(n(n-1)\) ordered pairs is an edge i.i.d. with prob \(p\). Adjacency matrix entries i.i.d. \(\text{Be}(p)\).

    Oriented Configuration Model \(\text{OCM}(d^+, d^-)\) (Cooper-Frieze '02)¶

    Each vertex \(i\) has \(d_i^+\) out-stubs and \(d_i^-\) in-stubs. Match out-stubs to in-stubs uniformly. Constraint: \(\ell_{\text{out}} = \sum d_i^+ = \sum d_i^- = \ell_{\text{in}}\).

    Oriented Preferential Attachment OPA (Bollobas-Borgs-Chayes-Riordan '03)¶

    Parameters \(\alpha + \beta + \gamma = 1\), "shifts" \(\delta_{\text{in}}, \delta_{\text{out}}\). At each time step: - \(\alpha\): new vertex creates link to old \(w\) with prob \(\propto (\delta_{\text{in}} + d_w^{\text{in}})\). - \(\gamma\): old vertex \(u\) chosen with prob \(\propto (\delta_{\text{out}} + d_u^{\text{out}})\) creates link to new vertex. - \(\beta\): link \(u \to w\) added with prob \(\propto (d_u^{\text{out}} + \delta_{\text{out}})(d_w^{\text{in}} + \delta_{\text{in}})\).

    Properties: OPA produces power laws for both in and out degrees, with correlated joint distribution.

    4. Bow-tie structure for \(D(n, c/n)\), \(c > 1\)¶

    Theorem. \(D(n, c/n)\) has a unique giant SCC of size \(\theta^2 n\) (where \(\theta\) = survival probability of \(\text{Poi}(c)\) BP), with: - SCC core: \(\theta^2 n\) - In-fan only (can reach SCC, not reachable from it): \(\theta(1-\theta) n\) - Out-fan only (reachable from SCC, can't reach it): \(\theta(1-\theta) n\) - Tubes / tendrils / disconnected: \((1-\theta)^2 n\) + small components \(O(\log n)\).

    Picture: 

       in-fan        SCC core        out-fan
   θ(1-θ)n       θ²n             θ(1-θ)n
      \         /    \         /
       ──────●──●──●──●──────

    5. Proof sketch¶

    BFS-out tree \(T_+(x)\) has same distribution as undirected BFS¶

    Each vertex has \(X_t \sim \text{Bin}(n - 1, p)\) out-neighbors; the BFS-out tree has the same distribution as the BFS tree for an undirected component in \(G(n, p)\) at the same \(p\). So $\(\Pr_{D(n,p)}(|C_+(x)| \ge a) = \Pr_{G(n,p)}(|C(x)| \ge a).\)$

    Lemma 1 (dichotomy)¶

    W.h.p. for all \(x\): either - Small fan-out: \(|C_+(x)| \le \frac{3c}{(c-1)^2}\log n\), OR - Large fan-out: \(|C_+(x)| \ge (\theta - \epsilon) n\).

    Same for \(C_-(x)\) (replace edges by reverse edges in the BFS).

    This dichotomy comes directly from the giant-component argument applied separately to forward and backward exploration.

    Define the "large" sets¶

    $\(L_+ = \{x : |C_+(x)| \ge \epsilon n\}, \quad L_- = \{x : |C_-(x)| \ge \epsilon n\}.\)$ \(E[|L_\pm|] \approx \theta n\) (matching the \(G(n,p)\) giant result).

    Lemma 3 (asymptotic independence of \(C_+, C_-\))¶

    For events \(A, B\) on \(C_+(x), C_-(x)\) respectively: $\(\Pr(A \cap B) = \Pr(A) \Pr(B)(1 + o(1)).\)$

    Why. BFS-out from \(x\) explores forward edges; BFS-in explores backward edges. Disjoint sets of edges → asymptotic independence (when not too many vertices are revealed).

    Putting it together¶

    • \(L_+ \cap L_-\) = vertices with both large fan-in and fan-out → all reach each other → strongly connected.
    • \(|L_+ \cap L_-| = \sum_x \Pr(|C_+(x)|, |C_-(x)| \text{ both large}) = \sum_x \theta \cdot \theta (1+o(1)) = \theta^2 n (1+o(1))\) by independence.
    • \(|L_- \setminus L_+|\) = in-fan only \(\approx \theta(1-\theta)n\).
    • \(|L_+ \setminus L_-|\) = out-fan only \(\approx \theta(1-\theta)n\).

    6. Method extraction¶

    • Method: Coupled forward/backward BFS + asymptotic independence between in- and out-components.
    • Recognition signal: "directed graph", "SCC", "bow-tie", "OCM", "OPA".
    • First move: identify \(C_+(x), C_-(x)\), apply giant component result to each separately, then multiply by independence.
    • Common mistake:
    • Treating SCC like an undirected component — it requires both directions of reachability, hence \(\theta^2\) not \(\theta\).
    • Forgetting the asymptotic independence — without Lemma 3, you'd get \(|L_+ \cap L_-| \le \min(|L_+|, |L_-|) = \theta n\), which overcounts.
    • Reusable lesson: Bow-tie sizes follow \((\theta(1-\theta), \theta^2, \theta(1-\theta), (1-\theta)^2)\) — exactly the joint distribution of two independent \(\text{Be}(\theta)\) variables.

    7. Connection to SIR¶

    The SIR digraph is exactly a directed graph in disguise: - Forward BP = exploration of \(C_+(x)\) (who gets infected starting from \(x\)) → outbreak probability \(P_+\). - Backward BP = exploration of \(C_-(x)\) (who could have infected \(x\)) → susceptibility probability \(P_-\).

    Different probabilities \(P_+ \ne P_-\) for SIR (because forward uses random \(T\) at parent, backward absorbs \(T\) into \(\bar p\)), but same threshold \(R_0 > 1\). Same algebra as the bow-tie picture here.

    8. Quick memory hooks¶

    • "Bow-tie: \(\theta(1-\theta), \theta^2, \theta(1-\theta), (1-\theta)^2\) — the four corners are products of \(\theta, 1-\theta\)."
    • "SCC = \(\theta^2\) because you need both forward AND backward survival."
    • "Forward and backward BFS are asymptotically independent — that's why it's a product."
    • "OPA: power laws on in AND out degrees, correlated."

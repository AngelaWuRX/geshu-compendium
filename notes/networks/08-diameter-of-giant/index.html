    Diameter of the Giant Component¶

    1. What this topic is about¶

    For \(G(n, c/n)\) with \(c > 1\), the diameter of the giant component (typical pairwise distance) is \(\Theta(\log n)\), with sharp constant \(1/\log c\). The proof has two halves: (a) lower bound via expected count of short paths, (b) upper bound via BFS exploration showing the queue grows like \(c^t\), then a "sprinkling" / connection step where two BFS trees from \(x\) and \(y\) meet.

    2. Core objects / definitions¶

    • Diameter (course convention): typical distance \(D = \text{dist}_{G(n,p)}(x,y)\) for \(x, y\) uniform from \(C_{\max}\).
    • BFS queue at generation \(a\): \(Z_a = Y_{t(a)}\) where \(t(a)\) is the time the last vertex of generation \(a-1\) leaves the queue.
    • Generation recurrence: \(t(a+1) = t(a) + Z_a\), with \(t(1) = 1\), \(Z_1 = X_1 = Y_1\).
    • Heuristic: \(Y_t \approx (c-1)t\), so \(Z_a \approx (c-1) c^{a-1}\), \(t(a) \approx c^{a-1}\).

    3. Main theorem¶

    Theorem. For \(c > 1\) in \(G(n, c/n)\): $\(\frac{D}{\log_c n} \to_P 1.\)$

    4. Proof structure¶

    Need to show for all \(\epsilon > 0\): - (a) \(\Pr(D \le (1-\epsilon) \log_c n) \to 0\) — diameter not too short. - (b) \(\Pr(D \ge (1+\epsilon) \log_c n) \to 0\) — diameter not too long.

    (a) Lower bound — expected path count¶

    Claim: For any \(x \ne y\) and \(\ell_0 = (1-\epsilon)\log_c n\): $\(\Pr_{G(n,p)}(d(x,y) \le \ell_0) \le \frac{1}{n} \cdot \frac{c^{\ell_0}}{1 - 1/c} = \frac{n^{-\epsilon}}{1 - 1/c}.\)$

    Proof. Let \(X_\ell = \#\text{paths } x \to y \text{ of length } \ell\). Then \(E[X_\ell] = (\#\text{paths}) \cdot p^\ell \le n^{\ell-1} p^\ell = c^\ell / n\). Sum geometric series.

    Conclusion. Compute: $\(\Pr(D \le \ell_0) \le \Pr(D \le \ell_0, |C_{\max}| \ge \theta n / 2) + \Pr(|C_{\max}| < \theta n / 2)\)$ where the first term is bounded by \((4/(\theta^2 n^2)) \sum_{x,y} \Pr(d(x,y) \le \ell_0) \le O(1/n) + O(n^{-\epsilon}) \to 0\).

    (b) Upper bound — BFS exploration meets at log¶

    Step 1. Bounds on \(t(a)\) and \(Y_t\) from concentration: - Claim 1: Set \(a_+ = (1 - 2\delta)\log_c n\). Then \(\Pr(t(a_+) \ge n^{1-\delta}) = O(n^{-\delta})\). (Markov on \(E[t(a)] \le c^a/(c-1)\).) - Claim 2: Set \(t_- = \delta \log_c n\), \(t_+ = n^{1-\delta}\). Then \(\Pr(|C(x)| \ge t_+ \text{ and } \exists t \in [t_-, t_+]: Y_t \le (c-1-\delta)t) = O(n^{-\alpha})\). (Concentration via SIR-type rate function on offspring sums.)

    Step 2. Assume both events: \(t(a_+) \le t_+\) and \(Y_t \ge (c-1-\delta)t\) for all \(t \in [t_-, t_+]\). Then iterating \(t(a+1) = t(a) + Y_{t(a)} \ge t(a) + (c-1-\delta)t(a) = (c-\delta)t(a)\) gives $\(t(a) \ge (c-\delta)^{a - a_-} t(a_-) \quad \text{for } a \in [a_-, a_+].\)$

    Step 3 (key claim). Set \(a_n = (\tfrac12 + \tfrac\epsilon3 + \delta)\log_c n\). Then for \(\delta\) small: $\(Z_{a_n} \ge n^{1/2 + \epsilon/4}.\)$

    Step 4 (meeting point). Run BFS from both \(x\) and \(y\) to depth \(a_n\). Both have \(\ge n^{1/2 + \epsilon/4}\) vertices in the frontier. Probability no edge between the two frontiers is \(\le (1-p)^{(n^{1/2+\epsilon/4})^2} \le e^{-p \cdot n^{1+\epsilon/2}} \to 0\).

    Step 5. Combined diameter \(\le 1 + 2 a_n = 1 + 2(\tfrac12 + \tfrac\epsilon3 + \delta)\log_c n \le (1+\epsilon)\log_c n\) for \(\delta\) small enough.

    5. Method extraction¶

    • Method: Generation recurrence \(t(a+1) = t(a) + Z_a\) + concentration on \(Y_t\) + "sprinkling" / two-BFS meeting argument.
    • Recognition signal: "diameter / typical distance in giant" → log scale, constant \(1/\log c\). Same shape applies to CM and PA.
    • First move: lower bound by counting paths (geometric series in \(c\)); upper bound by BFS exploration to depth \(\sim \frac12 \log_c n\) from both endpoints.
    • Common mistake:
    • Trying to bound \(D\) via single BFS to depth \(\log_c n\) — that's the whole diameter, but the meeting argument cuts depth in half (each side \(\frac12 \log_c n\)).
    • Forgetting \(|C_{\max}| \ge \theta n\) conditioning when picking uniform pair.
    • Reusable lesson: "Two BFS trees of size \(\sqrt n\) meet w.h.p." is the standard \(\frac12\) trick. It's what makes the constant \(1/\log c\), not \(1/\log c \cdot 2\).

    6. Quick memory hooks¶

    • "Diameter \(= \log_c n\)" — match generation count with population fraction.
    • "Each BFS tree fills to \(\sqrt n\), then they meet by birthday paradox."
    • "Generation recurrence \(t(a) \approx c^{a-1}\) — exponential growth at rate \(c\)."
    • For CM and PA the same \(\log n\) scaling holds; constant changes (use \(\nu\) instead of \(c\) for CM).

    7. Where else it appears¶

    • Lecture 14 (Small World): Watts-Strogatz / Newman-Watts beat the \(\log n\) scaling at constant clustering. Standard graph models still have \(\log n\).

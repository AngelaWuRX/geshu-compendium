    Expected Counts and Union Bounds¶

    1. What this topic is about¶

    Many proofs in random graph theory start by counting how many copies of some structure (cycles, isolated vertices, bad subgraphs, multi-edges) exist, then using expectation bounds to show the count is zero or nonzero w.h.p. The expected count method (first moment method) and the union bound are the simplest and most widely used tools in this course.

    2. Core objects / definitions¶

    Indicator sum: \(X = \sum_{i \in S} I_i\) where \(I_i = 1\) if structure \(i\) appears.

    First moment method: \(\Pr(X > 0) \leq E[X]\). If \(E[X] \to 0\), no structure exists w.h.p.

    Union bound: \(\Pr(\bigcup A_i) \leq \sum \Pr(A_i)\). (Same as first moment when \(X = \sum I_i\).)

    Second moment method: \(\Pr(X > 0) \geq \frac{(E[X])^2}{E[X^2]}\). Proves existence when the count concentrates.

    3. Standard proof moves¶

    1. Define \(X = \sum I_i\) — identify all candidate structures
    2. Count candidates — how many possible locations for the structure?
    3. Compute per-candidate probability — probability each candidate actually appears
    4. Multiply: \(E[X] = (\text{number of candidates}) \times (\text{probability per candidate})\)
    5. If second moment needed: compute \(E[X^2]\) by splitting into overlap cases

    4. Important lemmas / facts¶

    Expected number of cycles¶

    Statement: Expected number of \(k\)-cycles in \(G(n,p)\): $\(E[Y_k] = \binom{n}{k} \frac{(k-1)!}{2} p^k \leq (np)^k\)$

    Why it matters: Shows \(G(n,p)\) has no cycles when \(p \ll 1/n\).

    Proof: - Candidates: \(\binom{n}{k}\) ways to choose \(k\) vertices, \(\frac{(k-1)!}{2}\) unoriented cycles on them - Probability: \(p^k\) (all \(k\) edges present) - Total: \(\sum_{k \geq 3} E[Y_k] \leq \frac{(np)^3}{1-np} \to 0\) when \(np \to 0\)

    Expected number of trees of size \(k\)¶

    Statement: \(E[X_k] = \binom{n}{k} k^{k-2} p^{k-1}\)

    Useful bound: \(k^k \leq k! e^{k-1}\) (by induction using \((1+1/k)^k \leq e\)), giving: $\(E[X_k] \leq \frac{1}{epk^2}(npe)^k\)$

    Aggregate bound: \(E[X_{\geq k}] \leq pn^2 (npe)^{k-2}\), which is small for \(k \geq 2 + \frac{\log n}{\lambda_n}\) where \(\lambda_n = -\log(npe)\).

    Expected self-loops in configuration model¶

    Statement: Expected loops = \(\frac{1}{\ell_n - 1} \sum_i \binom{d_i}{2}\)

    Expected multi-edges in configuration model¶

    Statement: $\(\sum_{\{i,j\}} \Pr(A_{ij} \geq 2) \leq \frac{1}{2} \sum_{i \neq j} \frac{d_i(d_i-1) d_j(d_j-1)}{(\ell_n-1)(\ell_n-3)}\)$

    Proof idea: Bound \(1[A_{ij} \geq 2] \leq \frac{1}{2}A_{ij}(A_{ij}-1)\), expand using matching indicators, use disjoint pair probability.

    Wedge and triangle counts¶

    Statement: - Expected wedges: \(E[N_w] = 3\binom{n}{3}p^2 = \frac{n(n-1)(n-2)}{2}p^2\) - Expected triangles: \(E[N_\Delta] = \binom{n}{3}p^3\) - Global clustering: \(C_{\text{global}} \approx \frac{3E[N_\Delta]}{E[N_w]} = p\)

    Variance via overlap counting¶

    Statement: \(E[N_w^2]\) splits into three cases by edge overlap: - 2 shared edges (same wedge): contributes \(E[N_w] \cdot p^2\) (wait, it's just \(E[N_w]\) since same wedge) - 1 shared edge: \(n(n-1)(n-2)(2n-5)\) pairs, each needing \(p^3\) - 0 shared edges: at most \((E[N_w])^2 / p^4 \cdot p^4 = (E[N_w])^2\)

    Result: \(\text{Var}(N_w) \leq 9\left(\frac{1}{n^{3/2}p} + \frac{1}{n^3 p^2}\right)(E[N_w])^2\)

    5. Common mistakes¶

    • Overcounting cycles: forgetting to divide by \(2k\) (orientations and starting points) — result is \(\frac{(k-1)!}{2}\)
    • Not bounding \(\binom{n}{k}\) correctly: use \(\binom{n}{k} \leq \frac{n^k}{k!}\) for upper bounds
    • In variance calculations: missing an overlap case or miscounting the number of pairs with that overlap
    • Using \(E[X^2] = (E[X])^2\) (that's only true for constants!) — you need to actually compute \(E[X^2]\)
    • In the tree size argument: forgetting that \(E[X_{\geq k}]\) needs a geometric series sum, not just one term

    6. Quick memory hooks¶

    • "Count candidates, multiply by probability, sum — that's the expected count"
    • "\(\frac{(k-1)!}{2}\) for unoriented cycles: divide by \(k\) starting points and \(2\) orientations"
    • "Variance = how pairs of structures correlate, which depends on how much they overlap"
    • "Union bound is just first moment in disguise"
